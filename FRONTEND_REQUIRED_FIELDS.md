# Frontend Required Fields

This document contains only the required fields that the frontend must handle for each entity.

---

## ArticleDto (API Response)
- Id (Guid)
- Title (string)
- Content (string)
- Status (string)
- Language (string)
- CreatedAt (DateTime)
- CategoryId (Guid?)
- CategoryName (string?)
- CategorySlug (string?)

## VideoDto (API Response)
- Id (Guid)
- Title (string)
- Content (string)
- Status (string)
- Language (string)
- CreatedAt (DateTime)
- CategoryId (Guid?)
- CategoryName (string?)
- CategorySlug (string?)

## AudioDto (API Response)
- Id (Guid)
- Title (string)
- Content (string)
- AudioUrl (string)
- Status (string)
- Language (string)
- CreatedAt (DateTime)
- CategoryId (Guid?)
- CategoryName (string?)
- CategorySlug (string?)

## GalleryDto (API Response)
- Id (Guid)
- Title (string)
- Language (string)
- CategoryId (Guid)
- CategoryName (string)
- CategorySlug (string)
- Status (string)
- Items (List<PostItemDto>)

## SortedListDto (API Response)
- Id (Guid)
- Title (string)
- Language (string)
- CategoryId (Guid)
- CategoryName (string)
- CategorySlug (string)
- Status (string)
- Items (List<PostItemDto>)

## PostItemDto (API Response)
- Id (Guid)
- Title (string)
- ImageUrl (string)
- Content (string)
- PostId (Guid)

## CategoryDto (API Response)
- Id (Guid)
- Name (string)
- Language (string)
- Order (int)
- Layout (int)

## TagDto (API Response)
- Id (Guid)
- Name (string)
- Language (string)

## PageDto (API Response)
- Id (Guid)
- Title (string)
- Content (string)
- Language (string)
- Location (string)
- MenuOrder (int)

## UserDto (API Response)
- Id (Guid)
- UserName (string)
- Email (string)
- Role (string)
